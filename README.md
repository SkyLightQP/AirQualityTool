# AirQualityTool
온도, 습도 데이터를 입력받아 DB에 기록하고 시각적으로 표시합니다.

## 사용 부품

- NodeMCU (CH340)
- DHT11
- Grove DustSensor

## 참고 코드

- Dust Sensor Value -> ug/m^3

- http://wiki.seeedstudio.com/Grove-Dust_Sensor/
- https://blog.naver.com/issugroup/221004961500
